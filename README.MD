# Kuznechik GOST 34.12 R-2015 C# Implementation

## Features

 - This implementation uses ECB mode

This implementation was written for my bachelor's thesis about protecting Russian railways remote controls of railway traffic.

## How to compile

Just open the solution in MonoDevelop or Visual Studio
